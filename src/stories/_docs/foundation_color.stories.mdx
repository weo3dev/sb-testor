# Color

## Principles

### Color is meaningful

Color alone is not always enough to convey meaning, but it can help support the intended message. Use colors meaningfully to help users identify status, see actions, or indicate next steps.

### Color supports hierarchy

Color can be used to enhance and separate levels of important information. Use colors to draw a user's focus and attention to particular content areas.

### Color is accessible

Colors that meet standard contrast ratios provide accessible interfaces for a wide range of our users. Use accessible color combinations to increase usability and readability.
